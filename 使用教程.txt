========================================
  LLM with Toy - Bridge 桥接版
  安装与使用教程
========================================

【一、你需要准备什么】

1. 一台运行 AstrBot 的云服务器
2. 一台有蓝牙功能的 Windows 电脑（你自己的电脑）
3. Python >= 3.11（推荐 3.12），安装在你自己的电脑上
4. 体感反馈设备一个

========================================

【二、云端安装插件】

1. 将插件文件夹重命名为 astrbot_plugin_llm_with_toy
2. 放入 AstrBot 的 data/plugins/ 目录下
3. 重启 AstrBot
4. 看到日志显示 "[808 Plugin] 内置 Pull Server 已在 0.0.0.0:6013 启动" 即为成功
5. 确保云服务器防火墙已放行 6013 端口

========================================

【三、本地电脑安装 Bridge】

1. 打开命令提示符（cmd）
2. 安装依赖：
   pip install bleak aiohttp
3. 运行 Bridge 脚本：
   python bridge_808.py
4. 脚本会自动扫描蓝牙设备并连接

========================================

【四、蓝牙配对说明】

1. 长按设备电源键，指示灯闪烁 = 已进入广播模式
2. 确认电脑蓝牙已开启（Windows 设置 → 蓝牙和其他设备）
3. 运行 Bridge 脚本，它会自动扫描并连接，无需手动配对
4. 连接成功后控制台会显示设备名称和连接状态

【常见问题】
Q: 扫描不到设备？
A: 确认设备已开机且灯在闪；靠近电脑；重启蓝牙后重试

Q: 连接后经常断开？
A: 检查设备电量；减少电脑与设备之间的障碍物

Q: 附近有多个设备怎么办？
A: 默认连接信号最强的；可在脚本中指定设备 MAC 地址

========================================

【五、使用方式】

连接成功后，直接和 AI 聊天即可。
AI 会自动感知设备在线，根据对话内容自主控制设备。

你也可以直接说：
- "开一下设备"
- "调到 50"
- "慢慢加强"
- "关掉"

========================================
